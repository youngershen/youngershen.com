== Description ==

Automatically search for custom tag list, if the content of the article contains the text automatically add the article tags

自动搜索自定义的标签列表，如果文章内容包含该文本，则自动添加文章标签。
详情：[http://www.daxiawp.com/dx-auto-tags.html](http://www.daxiawp.com/dx-auto-tags.html)

== Installation ==

可以通过以下两种方法的其中一种来安装DX-seo插件：

1. 将下载的文件解压缩，然后将`dx-auto-tags`文件夹 上传到 `/wp-content/plugins/`目录，在插件后台启用

2. 直接在后台-安装插件，搜索'dx-auto-tags'，按照提示安装启用
